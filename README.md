# 环境 ：Node.js，Chrome

## 技术栈
```
Vue3.2 setup typescript
vue-router
Pinia
D3.js
```
## 项目安装
```
npm install
```

## 开发环境运行，支持热重载
```
npm run serve
```

## 项目打包以部署生产环境，打包后的文件位于dist目录下
```
npm run build
```

## 项目检查并自动修复潜在错误
```
npm run lint --fix
```
